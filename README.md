# Revenue_Performance_Analysis
[Link to dashboard](https://app.powerbi.com/view?r=eyJrIjoiNTQ5MjRjNjktYThmZC00NmIxLTkzMWYtYTA4MGVkNDQ0MjkwIiwidCI6ImVhODczMzkwLThjMWMtNDIzMS1hNzk5LTZiNWEwMjM1YjJlNiIsImMiOjN9&pageName=3f2e4fdec251587045a6)<br>
The purpose of this project is to identify regions product categories driving the majority of revenue from 2015-2018 The analysis focuses on revenue concentration and trends in revenue over time. It identifies the categories driving 60% of total revenue, as well as a steady upwards trend in total revenue over the timeline of the data. SQL was used to aggregate data and assist with time-based modeling, while DAX was used to assist with the charts in the dashboard.<br>
SQL is used for aggregating data to better recognize revenue drivers and trends.<br>
This analysis uses data sourced from a publicly available Kaggle dataset representing simulated sales data from 2015-2018, licensed under the GPL 2 license. The url of the original dataset is at https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting.
