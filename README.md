# Revenue_Performance_Analysis
The purpose of this dashboard is to identify product categories driving revenue and to recognize trends in revenue from 2015-2018.<br>
SQL is used for aggregating data to better recognize revenue drivers and trends.<br>
This analysis uses data sourced from a Kaggle dataset representing simulated sales data from 2015-2018, licensed under the GPL 2 license. The url of the original dataset is at https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting.<br>
